# Verificador de Fortaleza de Contraseña

Aplicación web desarrollada en React utilizando Vite. Permite verificar la fortaleza de una contraseña, mostrar u ocultarla, copiarla al portapapeles y generar contraseñas aleatorias con opciones avanzadas.

## Link de pagina subida en vercel: 
https://verificado-contrasenia.vercel.app/

## Funcionalidades principales

- Ingreso de contraseña
- Mostrar/Ocultar la contraseña
- Medición de fortaleza (Débil, Media, Fuerte)
- Copiar contraseña al portapapeles
- Mensaje de confirmación luego de copiar
- Generador de contraseñas aleatorias
- Opciones avanzadas:
  - Longitud personalizada
  - Incluir mayúsculas, minúsculas, números y símbolos
